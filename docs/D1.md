# D1 — Cloudflare D1 ORM Adapter

`github.com/tinywasm/goflare/d1` connects a Cloudflare D1 SQLite database to `tinywasm/orm` from within a Worker or Pages Function edge handler.

## Public API

```go
// New opens the named D1 binding and returns an *orm.DB.
// Returns ErrDatabaseNotFound if the binding is absent from the Worker env.
func New(bindingName string) (*orm.DB, error)

var ErrDatabaseNotFound error
```

`*orm.DB` exposes the standard tinywasm ORM methods: `Create`, `Query`, `Delete`, `CreateTable`, `DropTable`, `Close`.

## Usage

```go
//go:build wasm

package contact

import "github.com/tinywasm/goflare/d1"

func saveSubmission(form ContactForm) error {
    db, err := d1.New("DB") // "DB" matches the binding name in wrangler.toml
    if err != nil {
        return err
    }
    defer db.Close()

    if err := db.CreateTable(&ContactSubmission{}); err != nil {
        return err
    }
    return db.Create(&ContactSubmission{
        Nombre:  form.Nombre,
        Email:   form.Email,
        Mensaje: form.Mensaje,
    })
}
```

## Model setup

The model struct must implement `fmt.Model`. Use the `// ormc:form` directive and run `ormc` — it generates `ModelName()`, `Schema()`, `Pointers()`, and `Validate()` automatically:

```go
// ormc:form
type ContactSubmission struct {
    ID      int64
    Nombre  string
    Email   string
    Mensaje string
}
```

```bash
ormc
```

`ID int64` is auto-detected as primary key (auto-increment). No extra tags needed.

## wrangler.toml

```toml
[[d1_databases]]
binding      = "DB"
database_name = "my-db"
database_id   = "your-database-id"
```

## Constraints

- `d1` only compiles for `GOOS=js GOARCH=wasm` — all files carry `//go:build wasm`.
- Do **not** import `d1` from `web/server.go` or any non-wasm file.
- No `database/sql`. D1 is accessed exclusively via the D1 JS API.
- No stdlib `fmt`, `errors`, `strings` — use `github.com/tinywasm/fmt`.

## Architecture

```
orm.DB
 └── d1.adapter      (orm.Executor) — calls D1 JS API via AwaitPromise
 └── sqlt.Compiler   (orm.Compiler) — generates SQLite-dialect SQL
```

| Operation | D1 JS call |
|---|---|
| `Exec` | `db.prepare(sql).bind(...args).run()` |
| `QueryRow` | `db.prepare(sql).bind(...args).first()` |
| `Query` | `db.prepare(sql).bind(...args).raw({columnNames:true})` |

The D1 binding is read from `js.Global().Get("context").Get("env").Get(bindingName)`.
